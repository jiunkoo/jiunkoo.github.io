## 2. String 클래스
* 문자형 배열 참조변수(char[ ]) value를 인스턴스 변수로 정의
* 한번 생성된 String 인스턴스가 갖고 있는 문자열은 읽기만 가능(immutable, 변경 불가능)
* 문자열 리터럴은 컴파일 시 클래스 파일에 한 번만 저장
    * 클래스 파일이 클래스 로더에 의해 메모리에 올라갈 때 리터럴들은 JVM에 있는 상수 저장소(constant pool)에 저장
* 빈 문자열 저장 가능
